# Структурные блоки внутри контента окна

## Блок-модуль

### .config-module

Блок-контейнер с разделительной чертой снизу.

![](/assets/конфиг-модуль.jpg)

```
<div class="config-module">
    <--- КОНТЕНТ --->
</div>
```

### .label-block

</div>
